# Unsplash Clone

A modern Unsplash-inspired web application that allows users to search and explore high-quality images using the Unsplash API.

## Features

- 🔍 Search high-resolution images by keywords
- 🖼️ Responsive image gallery layout
- 📱 Fully responsive design for desktop and mobile
- 📥 View and explore images in detail

## Tech Stack

- HTML
- CSS
- JavaScript
- Unsplash API
- Vercel (Deployment)

## About the Project

This project is an Unsplash-inspired image browsing platform built for learning purposes.

## What I Learned

- Working with APIs and fetching data
- Handling asynchronous JavaScript (`async/await`)
- Building responsive UI layouts
- Managing search functionality
- Deploying applications using Vercel
